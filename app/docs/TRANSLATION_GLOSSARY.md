## Usage Guidelines

### 1. Consistency
Always use the exact translation from this glossary. Do not create variations.

**✅ Correct:**
- EN: "Enable Plugin" → ES: "Activar Plugin"

**❌ Incorrect:**
- EN: "Enable Plugin" → ES: "Habilitar Plugin" (inconsistent)

### 2. Context Awareness
Some terms may need slight adjustments based on grammatical context, but the core term should remain consistent.

**Example:**
- "Plugin enabled" → "Plugin aktiviert" (DE), "Plugin habilitado" (ES), "Plugin activé" (FR)

### 3. Brand Names
Keep brand names and product names unchanged:
- TikTok Live
- WebGPU
- VDO.Ninja
- OpenShock
- HybridShock

### 4. Acronyms
Common acronyms remain unchanged across all languages:
- TTS (Text-to-Speech)
- API (Application Programming Interface)
- OSC (Open Sound Control)
- HUD (Heads-Up Display)
- GCCE (Global Chat Command Engine)

### 5. UI Constraints
When translating for UI elements, be mindful of:
- Button width limitations
- Tooltip space
- Menu item length
- Modal dialog titles

If a translation is too long, use the shorter alternative or abbreviation provided in this glossary.

## Maintenance

This glossary should be updated whenever:
- New features are added
- New plugins are created
- Terminology changes are required
- Inconsistencies are found

All updates must maintain consistency across all four languages (DE, EN, ES, FR).
