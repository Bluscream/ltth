🍞 Toaster Mode (Low-End PCs)

Uses Canvas 2D rendering instead of WebGL for maximum compatibility

🔧 What changes:
• Renderer: WebGL → Canvas 2D
• GPU Acceleration: Disabled
• Compatibility: Maximum
• Performance: Reduced but stable
• Visual Effects: All maintained
💡 Tip:

Use Toaster Mode if your PC slows down during fireworks or if you experience rendering issues in OBS.

⏱️ Queue System (Lag Prevention)

Limit the number of fireworks per second to prevent lag during gift spam. When enabled, excessive fireworks are skipped to maintain smooth performance.

1/s (Strict) 5/s 20/s (Relaxed)

Lower values prevent more lag but skip more fireworks during spam

⚠️ Note:
• When rate limit is reached, additional fireworks are skipped (not queued for later)
• Recommended: 5-10 rockets/sec for balanced performance
• Lower settings help prevent stuttering during extreme gift spam
• Disable this feature if you prefer to see every firework regardless of performance
