Frequently Asked Questions

Why do I need to restart after switching profiles?

The profile system loads your database and settings once at startup. When you switch profiles, the new profile information is saved, but the application continues using the old profile's database until you restart. This prevents data corruption and ensures a clean transition.

Will my viewer XP data persist after updates?

Yes! All profile data is stored in a persistent location outside the application directory. On Windows: %LOCALAPPDATA%\pupcidslittletiktokhelper\user_configs\. This location survives all application updates.

Is viewer XP data shared between profiles?

No! Each profile has its own completely independent database file. Viewer "alice" in profile "streamer1" is completely separate from viewer "alice" in profile "streamer2". They each have their own XP, levels, and badges.

Can I enable auto-restart after profile switch?

Yes! In the User Profiles section above, you can enable the "Auto-Restart on Profile Switch" checkbox. When enabled, the application will automatically restart 5 seconds after switching profiles, with a countdown you can cancel if needed.
