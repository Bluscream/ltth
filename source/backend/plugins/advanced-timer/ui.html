Help & Documentation

🎯 Timer Modes

Countdown: Counts down to zero – great for subathon timers

Count Up: Counts up to a target value

Stopwatch: Counts up indefinitely

Loop: Continuously repeats countdown

Interval: Counts up to target, then resets and repeats

⚡ Direct Interactions

Set Per Coin, Per Follow, Per Share, Per Subscribe, Per Like, Per Chat Message directly on each timer card. Use positive values to add time, negative to reduce.

Enable the x Multiplier to multiply all interaction values by a factor (e.g. ×10 for events).

🔗 Advanced Event Rules

Need gift-name filters or chat-command matching? Use the Advanced Event Rules link on each timer card to configure per-gift / per-command rules.

📊 OBS Overlay

Each timer card shows its overlay URL. Add it as a Browser Source in OBS.
