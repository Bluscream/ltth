# Historical Documentation Archive

This directory is intentionally not part of the active documentation path.

Files here are old implementation reports, migration experiments, fix summaries, and planning notes. They are useful for context, but they may contradict the current snapshot.

Before using anything from this archive:

1. Check `README.md`, `AGENTS.md`, `DOCUMENTATION_INDEX.md`, and `docs/SNAPSHOT_STATUS.md`.
2. Verify the claim against current source code.
3. Promote only the still-relevant part into active docs if needed.

Do not update archive files during normal feature work.
